# 🎓 Student Management System in C++

A command-line Student Management System developed in **C++**. This project demonstrates object-oriented programming, file handling, and efficient student record management through a simple and interactive console application.

## ✨ Features
- ➕ Add student records
- 📋 View student details
- 💾 File handling for data storage
- 🖥️ User-friendly command-line interface

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

## ▶️ How to Run

1. Clone this repository:
```bash
git clone https://github.com/vishalrana45/studen_management_system_cpp.git
```

2. Compile the program:
```bash
g++ student_management_system.cpp -o student
```

3. Run the program:
```bash
./student
```
